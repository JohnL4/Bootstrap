Bootstrap attributes

role
This is an ARIA attribute with a TON of possible values. See the W3C spec.
data-toggle
The HTML5 spec allows arbitrary attributes whose names start with data- and continue with any XML-compliant string. This is the "custom data" attribute, and Bootstrap uses data-toggle in a variety of ways, such as "modal", "tab", "collapse", "dropdown". There isn't really a single place (as far as I know) where all of data-toggle is documented; it's just sort of scattered.
